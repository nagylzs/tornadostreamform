
Welcome to tornadostreamform's documentation!
=============================================


Apart from providing the package itself, this project comes with two
test web servers that you can try. The test web servers are well
documented files, and you can get the know-how from there. Please note
that the test server is not included in the pypi package (e.g. it
cannot be installed using pip or easy_install), but you can download
the whole project repository from here:
https://bitbucket.org/nagylzs/tonadostreamform/downloads or you can
clone the project. For details, see https://bitbucket.org/nagylzs/tonadostreamform
